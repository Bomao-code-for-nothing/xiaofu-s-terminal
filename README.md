## 安裝教學

1. 從 GitHub 取得小芙終端機

```bash
git clone https://github.com/Bomao-code-for-nothing/xiaofu-s-terminal
cd xiaofu-s-terminal
```

2.執行安裝腳本
```bash
./install.sh
```
3.大功告成，重啟終端機讓指令生效
```bash
cd ~/
source ./.zshrc
```

## 如何解除安裝
方法1.**修改.zshrc**
```
open ./
按下：Command+shift+.
右鍵，以任意文字編輯器打開，把小芙終端機的部分刪除
```
